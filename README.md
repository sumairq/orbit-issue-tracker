# Jira-Clone Application

> This is a clone of the popular Bug tracker application i.e Jira . It is built with React for client side and Node.js for the server and Postgres for
database. 


## Built With

- JavaScript, TypeScript, HTML, CSS, and SQL
- React, Node, Express, Postgres, TypeORM, Cypress.io, webpack
- Eslint, Styled Components

## Getting Started

To get a local copy up and running follow these simple example steps.

### Prerequisites

- Install postgreSQL and create a database named `jira-clone_development`.

### Setup

- `git clone https://github.com/sumairq/jira-clone.git `


- App should now be running on `http://localhost:8080/`

### Install

- `npm run install-dependencies`

### Usage

- `cd api && npm start`
- `cd client && npm start` in another terminal tab

### Run tests

- Set up development environment
- Create a database named `jira_test` and start the api with `cd api && npm run start:test`
- `cd client && npm run test:cypress`
